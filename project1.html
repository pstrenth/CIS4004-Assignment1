Document Comparison Service

Type: Internship / Professional Project

Tech: C#, ASP.NET Core, APIs

Overview

A backend service that accepts two documents via multipart upload and returns structured differences, enabling users to compare document versions efficiently.

Key Contributions

  • Built an API controller endpoint for file uploads (multipart/form-data)
  • Validated inputs and handled large file sizes safely
  • Integrated service layer logic to process comparisons
  • Designed for secure access via role-based authorization

What I Learned

  • Service/controller separation and clean architecture
  • Handling file upload validation and constraints
  • Security patterns (authorization roles)
