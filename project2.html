Enterprise Financial Data Dashboard

Organization: BNY Mellon

Role: Software Engineer Intern

Duration: June 2025 – August 2025

Technologies: Angular, Spring Boot, SQL, Tableau

Project Overview

During my Software Engineering Internship at BNY Mellon, I contributed to the development of an enterprise financial data dashboard used by internal teams to securely view and interact with real-time financial information while meeting regulatory compliance standards.

Key Contributions

  • Developed and deployed Angular-based user interfaces that are now live in production.
  • Integrated frontend components with a Spring Boot backend and SQL database.
  • Implemented secure file upload functionality with validation to ensure data integrity.
  • Worked with Tableau dashboards to visualize financial data for internal stakeholders.
  • Collaborated with senior engineers and product stakeholders in Agile sprints.

What I Learned

  • Enterprise-scale application architecture and clean separation of concerns.
  • Secure data handling practices in regulated financial environments.
  • Working in Agile teams and delivering production-ready features on tight timelines.
  • Importance of compliance, validation, and auditability in financial systems.
